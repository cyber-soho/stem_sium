Standardized Assessment Framework Proposal

Addressing the gap in current standardization for practical courses. This framework proposes a hands-on, rubric-based assessment model tailored specifically for the Integrative Project sequence.

Dec 9, 2025
To: Academic Leadership

Key Objectives

  • Develop common grading standards.
  • Accommodate practical components.
  • Define Midterm/Final weights.
